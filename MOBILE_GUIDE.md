# Mario Platformer - Mobile Gaming Guide

Welcome to the mobile version of the Mario Platformer! This guide will help you get the most out of the touch controls and mobile features.

## 🎮 Getting Started on Mobile

### Device Requirements
- **iOS**: Safari 12+, Chrome 60+
- **Android**: Chrome 60+, Firefox 55+, Samsung Internet
- **Screen Size**: Works on phones (4"+ recommended) and tablets
- **Performance**: Modern devices from 2018+ for best experience

### First Time Setup
1. Open the game in your mobile browser
2. Add to home screen for full-screen experience:
   - **iOS**: Tap Share → Add to Home Screen
   - **Android**: Tap Menu → Add to Home Screen
3. For best performance, close other browser tabs
4. Enable landscape mode for larger controls (optional)

## 🕹️ Touch Controls Overview

### Virtual D-Pad (Left Side)
- **Left Arrow**: Move Mario left
- **Right Arrow**: Move Mario right  
- **Up Arrow**: Alternative jump button
- **Down Arrow**: Crouch (Big/Fire Mario only)

### Action Buttons (Right Side)
- **Green JUMP Button**: 
  - Tap: Small jump
  - Hold: Higher jump (variable height)
  - Works like spacebar on desktop
  
- **Orange RUN Button**:
  - Hold while moving: Run faster
  - Hold + Jump: Jump further
  - Essential for reaching high platforms
  
- **Red 🔥 Button**:
  - Only appears when you're Fire Mario
  - Tap to shoot fireballs
  - Automatically hidden for Small/Big Mario

### Utility Buttons (Top Right)
- **⏸️ Pause**: Pause/resume the game
- **🐛 Debug**: Toggle collision boxes and debug info

## 🎯 Gameplay Tips for Mobile

### Movement Mastery
- **Smooth Running**: Hold RUN button, then press direction
- **Precise Jumping**: Tap JUMP for small hops, hold for big jumps
- **Wall Jumping**: Jump toward wall, quickly reverse direction
- **Quick Turns**: Release direction, tap opposite direction

### Power-Up Strategy
- **Small Mario**: Focus on safe jumping and collecting mushrooms
- **Big Mario**: Use RUN button for breaking blocks and longer jumps
- **Fire Mario**: Combine running and fireball shooting for maximum effectiveness

### Advanced Techniques
- **Coyote Time**: Brief moment after leaving platform where you can still jump
- **Variable Jumping**: Hold jump longer for higher platforms
- **Speed Control**: Use RUN button selectively to control speed
- **Fireball Timing**: Fire diagonally down for maximum enemy coverage

## 🔧 Troubleshooting

### Performance Issues
- **Close Other Apps**: Free up system memory
- **Reduce Browser Tabs**: Limit to just the game
- **Check Battery**: Low battery can throttle performance
- **Restart Browser**: Clear cache and restart if needed

### Control Problems
- **Calibrate Touch**: Ensure screen is clean and dry
- **Check Orientation**: Some features work better in landscape
- **Update Browser**: Use latest browser version
- **Disable Zoom**: Make sure browser zoom is at 100%

### Common Issues

#### "Controls Don't Respond"
- Clean your screen
- Ensure you're tapping the button areas
- Try reloading the page
- Check if another app is overlaying

#### "Game Runs Slowly"
- Close background apps
- Reduce browser tabs
- Check available storage space
- Try a different browser

#### "Can't Reach High Platforms"
- Hold RUN button while jumping
- Try running start for momentum
- Use variable jump height (hold JUMP longer)
- Look for alternative paths

#### "Fireball Button Missing"
- You need to be Fire Mario (collect Fire Flower)
- Button only appears in Fire power state
- Get Super Mushroom first, then Fire Flower
